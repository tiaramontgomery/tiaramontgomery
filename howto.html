How-to Website

A How-to website designed to teach you how to use gym equipment and allow you to create your own workout plan. This website is dedicated to guiding you through the world of gym equipment and empowering you to create your own personalized workout plan.

Problem

First time gym-goers not knowing how to use machines and structure workouts

Going to the gym for the first time is intimating!

Solution

Users can search and learn how to use the equipment

  • Users search through different muscle groups and find popular machines in each.

Users can add to their workout plan

  • Users can customize their workout plan with how many sets and reps.

Responsive Design

Conclusion

This was a design project in my first UI/UX design course at the University of Miami. I was tasked with creating a responsive how-to application. I decided to do a how to use gym equipment website since I enjoy going to the gym. During this project, I continued to familiarize myself with Figma and how to make a responsive web version. Looking back, I would love to reimprovise the design as well as conduct user interviews to see what the application is missing and how to improve it in the future.
